# janus

> Cloudflare worker as a reverse proxy

## Usage

Define a `.dev.vars` file with the populated contents of `.dev.vars.sample`.

For local testing:

```console
npx wrangler dev
```

Deployment:

```console
npx wrangler deploy
```
